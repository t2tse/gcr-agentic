# Stash API

The backend service for Stash, a link management and bookmarking application.

## Description

Stash API is built with NestJS and provides link storage and organization capabilities. The backing database is Firestore.

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
