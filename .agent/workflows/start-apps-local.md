---
description: Start all apps locally in this code base.
---

1. Start the Checkmate service (NestJS)
// turbo
2. cd checkmate && npm run start:dev

3. Start the Stash service (NestJS)
// turbo
4. cd stash && npm run start:dev

5. Start the Portal frontend (Next.js)
// turbo
6. cd portal && npm run dev

7. Start the Todo Agent (ADK)
// turbo
8. cd todo-agent && make local-backend

9. Start the Personal Assistant Agent (ADK)
// turbo
10. cd personal-assistant-agent && make local-backend
